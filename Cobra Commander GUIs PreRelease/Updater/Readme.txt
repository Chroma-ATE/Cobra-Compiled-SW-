Copy the Updater folder and its contents into the GUI executable folder.

Once you're done, the folders and files should have the following structure:
 
Necessary files
----------------------
C:\Program Files (x86)\GUI 3.0\Cobra GUI.exe
C:\Program Files (x86)\GUI 3.0\Updater\Updater.exe


Optional files (You don't have to copy the following files.  They are autogenerated when required)
-----------------------
C:\Program Files (x86)\GUI 3.0\Cobra GUI.aliases
C:\Program Files (x86)\GUI 3.0\Cobra GUI.ini
C:\Program Files (x86)\GUI 3.0\Updater\Updater.ini
C:\Program Files (x86)\GUI 3.0\Updater\Updater.aliases
